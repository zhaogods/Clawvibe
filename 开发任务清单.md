# 开发任务清单

## 完成目标

4 小时内完成一个接真实数据流的 OpenClaw 排障助手。

## 必做任务

### 1. 搭目录

- 创建 `app/`
- 创建 `runs/`
- 创建 `examples/`

### 2. 写主入口

- 新建 `app/main.py`
- 支持输入：
  - 仓库地址
  - 报错日志
  - 触发步骤

### 3. 写仓库扫描

- 新建 `app/repo_reader.py`
- 识别：
  - Python / Node.js
  - 依赖文件
  - 配置文件
  - 入口文件

### 4. 写报错分析

- 新建 `app/analyzer.py`
- 输出：
  - 错误类型
  - 简明解释
  - 关键词

### 5. 写复现建议

- 新建 `app/reproducer.py`
- 输出：
  - 实际执行命令
  - 运行结果
  - 排查顺序

### 6. 写报告输出

- 新建 `app/reporter.py`
- 输出 `final_report.md`

### 7. 准备样例

- 准备 1 个真实 Python 报错案例
- 准备 1 个真实 Node.js 报错案例

## 不做任务

- 不做登录
- 不做数据库
- 不做复杂前端
- 不做自动提交 PR
- 不做 mock
- 不做假数据回放
- 不做静态占位结果

## 验收标准

- 输入一个案例后，能生成一份结构化报告
- 报告中至少包含：
  - 报错解释
  - 关键文件
  - 真实复现结果
  - 修复建议
  - 验证步骤
