# 任务计划

## 目标

在半天内完成一个可运行的小项目方案，主题为：

**让新手也看懂报错：我用 OpenClaw 做了个会读仓库、会复现、会给补丁的 vibe coding 助手**

目标不是做大平台，而是完成一条可演示闭环：

- 输入仓库地址、报错日志、触发步骤
- 输出报错解释、仓库摘要、相关文件、复现建议、修复建议

## 范围约束

- 只做 Python / Node.js 仓库的基础识别
- 只覆盖依赖缺失、配置错误、启动失败三类问题
- 不做复杂前端
- 不做自动提交代码
- 不做 mock、假数据和占位结果
- 第一版必须接真实仓库、真实日志和真实执行结果

## 风险

- 时间只有半天，不能把“自动复现 + 自动补丁 + UI + 云部署”全部做深
- 最合理的完成标准是：先把主链路做通，再补界面和展示材料
